# 💖 It Pink Club

Plataforma de achadinhos desenvolvida para organizar e divulgar produtos selecionados.

## 🌐 Site

Produção:
https://pinkclub-three.vercel.app

## ✨ Funcionalidades

- Cadastro de produtos
- Edição de produtos
- Exclusão de produtos
- Pesquisa em tempo real
- Filtros por categoria
- Painel administrativo
- Layout responsivo
- Integração com Firebase
